# InfluenceOS

## Vision

InfluenceOS is an AI-powered influencer marketing platform that helps agencies and brands discover creators, manage campaigns, organize influencer relationships, and automate outreach.

## Version 1 Goals

- Dashboard
- Influencer CRM
- Search & Filters
- Campaign Manager
- Client Portal

## Future Goals

- Instagram Discovery
- TikTok Discovery
- AI Categorization
- AI Outreach
- Reporting
- Public API
- SaaS Subscriptions

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- TanStack Query
- Cloudflare Workers
- Cloudflare D1
- Cloudflare R2
- Drizzle ORM
